Dados Sensíveis

De acordo com o filme Snowden, é possível entender que a coleta de dados de sites é algo muito sensível e que precisa de muito cuidado, pois pode comprometer gravemente a privacidade dos usuários. No filme, o personagem decide vazar informações de ações de invasão de privacidade do governo norte-americano, pois ele entende que os métodos que são usados são imorais. Entende-se que, quando é necessário coletar alguma informação, deve-se informar de forma explícita para qual finalidade será, sempre respeitando a Lei Geral de Proteção de Dados. Formulários que coletam dados devem ser feitos com atenção para não armazenar dados desnecessários. Informações como CPF, e-mail e senha devem ser tratadas com muito cuidado, pois são extremamente comprometedoras. Campos de entrada devem utilizar tipos apropriados, como (type="password") para senhas, e nunca permitir preenchimento automático (autocomplete="off") em dados sigilosos. Além disso, a comunicação com o servidor deve ser sempre feita via HTTPS, garantindo que os dados trafeguem de forma criptografada.

  • Utilizar conexões HTTPS em todo o sistema
  • Exibir banner de consentimento para coleta de dados
  • Não armazenar senhas ou tokens em localStorage
  • Informar de forma clara a finalidade dos dados coletados
  • Reduzir a coleta de dados ao mínimo necessário
  • Usar campos tipo "password" para senhas, nunca "text"
  • Evitar autocomplete para campos sensíveis
